# Gesture Controlled Car using ESP32

Built a real-time gesture-controlled robotic car using dual ESP32 boards and MPU6050 sensor.

## Features
- Gesture-based control using accelerometer + gyroscope
- Wireless communication using ESP-NOW
- Real-time motor control using L298N driver
- Supports forward, backward, left, right, stop

## Tech Stack
- ESP32
- MPU6050
- ESP-NOW
- Arduino C++

## Demo
https://youtu.be/ekI203TfxyY

## How it works
Hand movements are captured using MPU6050, processed by ESP32, and transmitted wirelessly to control the car in real time.
